ПОЛЕЗНОЕ-VSCode
===============
 Оборачивание тегом выделенной области:
 *Alt + W*

Закладки:
---------
F1 и выбрать для того чтобы поставить закладку:
 *Bookmark toggle*
F1 и выбрать, чтобы посмотреть список закладок: 
 *Bookmark list*  

Поставить закладку с номером:
 *Ctrl + Shift + 1, 2, 3...*
 *Ctrl + 1, 2...*

Пустые строки:
--------------
Удалить пустые строки:
 *Ctrl + ;*
!_Если набрать "blank" в настройка и убрать галочку, то удалять
будет все пустые строки

Переместить строку вверх\вниз:
-----------------------------
*Alt + Стрелка Вверх\Вниз*

Перейти из HTML к классу CSS:
---------------------------- 
*Клавиша "F12"*

Запустить Терминал:
-------------------
*Ctrl + ~ *

Свернуть весь код:
------------------
*Fold All: CTRL+K, CTRL+0 (zero)

Fold Level [n]: CTRL+K, CTRL+[n]*

Unfold All: CTRL+K, CTRL+J

Fold Region: CTRL+K, CTRL+[

Unfold Region: CTRL+K, CTRL+]

SASS/SCSS-
==========
1. Можно установить правила для компиляции Глобально,
можно установить для рабочего проекта. Для этого в папке
.vscode создать файл settings.json с настройками для этого проекта (https://github.com/ritwickdey/vscode-live-sass-compiler/blob/master/docs/faqs.md).
2. Чтобы в каталоге sass не компилироваться файл необходимо добавить подчеркивание, например "_файл.scss.
3. чтобы этот файл попал в главный style.css необходимо добавить в style.scss директиву: '@import "_файл/scss";
'. Тогда стили будут компилироваться в главный style.css.   

Git и Github
============
Как выйти из git log?
--------------------
* Q * проверь раскладку !!
* Ctrl + C *
* :q *
* :z *
* Ctrl + Z *
Универсальный способ: ESC,Shit+Z,Shit+Z



Как выйти из редактора git bush?
--------------------------------
После git commit команды вы вошли в редактор: 
- поэтому сначала нажмите, i (режим ввода текста)
- затем начните печатать. 
- После совершения вашего сообщения (переходим в командный режим) нажмите Esc или Ctrl + c 
- затем :wq
